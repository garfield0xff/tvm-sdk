# TVM SDK

C++ SDK for TVM (Apache TVM) 

## Prerequisites

- **Docker**

## Quick Start

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd tvm-sdk
```

### 2. Build Docker image

**Linux/macOS:**
```bash
./script/setup-docker.sh
```

**Windows (PowerShell):**
```powershell
.\script\setup-docker.ps1
```
